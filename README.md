Smart Hydration Tracker

Overview

Smart Hydration Tracker is a simple web application built using Streamlit. It helps users track their daily water intake and maintain proper hydration habits.

Features

- Add daily water intake
- Track total water consumed per day
- Date-wise tracking
- Reset daily progress
- Simple and user-friendly interface

Tech Stack

- Python
- Streamlit
- Pandas
- datetime module

Installation and Setup

1. Clone the repository
git clone https://github.com/your-username/smart-hydration-tracker.git

2. Go to project folder
cd smart-hydration-tracker

3. Install dependencies
pip install streamlit pandas

4. Run the application
streamlit run app.py

How It Works

1. User opens the application
2. Enters water intake value
3. Data is stored and updated
4. Total daily intake is displayed
5. Helps users maintain hydration goals
